--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
body는 상체 제어 모듈임. 총 두개의 클럭사용(1초, 짧은 것)
input: neck, body, head
clock(일반적인 클럭), clock_1sec(주기가 1초인 클럭)
high(1 INPUT), low(0 INPUT)

output: neck_output(목유압)(1작동, 0멈춤), head_output(머리 두드리기)(1작동,0멈춤), body_output(몸 두드리기)(1작동, 0멈춤), 
rail_move(레일 움직이기)(1레일 상승, 0레일 하강, 정지는 1,0반복), rail_output(레일 지압 가동 및 레일 작동)(1작동, 0멈춤)

neck == 1 이면 레일을 목끝까지 올려 (목끝<->목시작) 반복해서 움직임.(움직이면서 레일 지압 가동), 목 유압 가동
body == 1 이면 레일을 충분히 하강 후 (가장아래 <->목시작) 반복해서 움직임.(움직이면서 레일 지압 가동), 몸 두드리기 가동
neck == 1 && body == 1 이면  레일 충분히 하강 후 (가장아래 <->목끝) 반복해서 움직임.(움직이면서 레일 지압 가동), 몸 두드리기 가동, 목 유압 가동

mj_head_hit 는 머리 두드리는 모듈 제어 모듈임. 1초짜리 클럭 하나 사용

1		0	 (머리 가장 윗부분, 뒷통수 정수리 약간 아래) (S0)
2	           0    0           (S1,  S2)
3	         0        0         (S3, S4)

3> 2 -> 1 -> 3-> 2-> 1 순으로 한 번씩 두드리면서 반복함(모듈을 1초 때리고 1초 올리고)
input: ON(1)/OFF(0), clock_1sec(주기가 1초)
output: S0, S1, S2, S3, S4 (1작동, 0멈춤)
